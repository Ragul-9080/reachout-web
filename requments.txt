3. Pages & Features
Public Website
•	Home – Overview of Reachout Academy
•	Courses – List of all courses (dynamic from DB)
•	Verify Certificate – Enter certificate number → Show student details
•	About Us – Mission, team
•	Contact Us – Form + location
Admin Dashboard
•	Login – Secure authentication
•	Manage Courses
o	Add new course (title, description, image, fees, duration)
o	Edit course
o	Delete course
•	Manage Certificates
o	Add certificate details (student name, course, issue date, cert number)
o	Delete certificate
•	View All Certificates
•	Logout

4. Database Design
Courses Table
Field	Type	Description
id	String	Unique ID
title	String	Course Name
description	String	Details
duration	String	Duration (e.g., 3 months)
fees	Number	Course Fee
image	String	Course Image URL
Certificates Table
Field	Type	Description
id	String	Unique ID
studentName	String	Student's Name
courseName	String	Course Completed
issueDate	Date	Date of Issue
certNumber	String	Unique Certificate Number
status	String	Valid / Expired
Admin Users Table
Field	Type	Description
id	String	Unique ID
email	String	Admin Email
password	String	Hashed Password
